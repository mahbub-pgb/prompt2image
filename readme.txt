Generate AI-powered images directly from text prompts in your WordPress admin panel.

== Description ==

Prompt2Image allows you to generate images from text prompts using AI, directly inside WordPress.  
Perfect for content creators, bloggers, marketers, and developers who want to create visuals on the fly.

**Features:**
* Generate AI images from text prompts.
* Save images to the WordPress Media Library.
* Export image data for integration with other systems.
* Modern, user-friendly admin interface.
* Secure with nonce verification and user role permissions.

== Installation ==

1. Download the `prompt2image.zip` plugin file.
2. In WordPress admin, go to **Plugins → Add New → Upload Plugin**.
3. Upload `prompt2image.zip` and click **Install Now**.
4. Activate the plugin.
5. Navigate to **Prompt2Image** in the admin menu to start generating images.

== Frequently Asked Questions ==

= Do I need an API key? =
Yes, you must provide a valid API key to connect to the image generation server.

= Can I use this for commercial projects? =
Yes, generated images can be used commercially, following the terms of your AI service provider.

= Which user roles can generate images? =
Only users with appropriate capabilities (administrator/editor) can generate images.
